About This Project

Overview

This web application performs AI-based voice call sentiment analysis using deep learning. Speech audio is analyzed to detect emotions, which are then mapped to Positive, Neutral, or Negative sentiment categories.

Dataset

Trained on the RAVDESS (Ryerson Audio-Visual Database of Emotional Speech and Song) dataset. Emotion labels are parsed automatically from filenames.

Features
  • MFCC (40 coefficients)
  • Chroma (12 bins)
  • Mel Spectrogram (40 pooled bands)
  • Zero Crossing Rate
  • RMS Energy
Model

1D Convolutional Neural Network (CNN) built with TensorFlow/Keras. Architecture: Conv1D → BatchNorm → Pooling → GlobalAveragePooling → Dense.

Tech Stack

Python, Flask, TensorFlow, Librosa, scikit-learn, Bootstrap 5, Chart.js, SQLite, ReportLab (PDF reports).

Project Structure

voice_sentiment_analysis/
├── app.py              # Flask web server
├── train_model.py      # Model training pipeline
├── predict.py          # Inference utilities
├── audio_features.py   # Librosa feature extraction
├── config.py           # Settings & mappings
├── database.py         # SQLite integration
├── dataset/            # RAVDESS audio files
├── saved_model/        # model.h5 & artifacts
├── static/             # CSS, JS, uploads, plots
└── templates/          # HTML pages

Setup

  1. pip install -r requirements.txt
  2. Download RAVDESS and extract into dataset/
  3. python train_model.py
  4. python app.py
